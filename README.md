# ContribLens

Personal GitHub contribution analytics ‚Äî exposing what your commits are actually made of.

Embed a live stats card in your GitHub README in one line:

```markdown
![ContribLens](https://your-deploy.com/svg/your-username)
```

## What it does

Given a GitHub username, ContribLens:

1. Fetches your **public contribution history** from the GitHub API
2. Applies **contribution rules** ‚Äî ignores generated files, lock files, vendor code, minified files, rename-only commits, etc.
3. Calculates your **estimated personal language usage** from the files you actually authored
4. Returns the result as **JSON** or a **SVG card** ready to embed

## Endpoints

| Endpoint | Description |
|---|---|
| `GET /analytics/:username` | Full analytics snapshot as JSON |
| `GET /svg/:username` | SVG card for embedding |
| `GET /health` | Health check |

### Query parameters

Both endpoints accept:

| Param | Default | Description |
|---|---|---|
| `window` | `1` | Years of history to analyze (`1`‚Äì`5`, or `all`) |
| `theme` | `default-dark` | Card theme (`default-dark`, `default-light`, `high-contrast`) |

### Example

```
GET /svg/torvalds?window=1&theme=default-dark
GET /analytics/torvalds?window=all
```

## Self-host

### Requirements

- Node.js ‚â• 22
- A GitHub personal access token with `read:user` and `public_repo` scopes

### With Docker

```bash

docker run -e GITHUB_TOKEN=ghp_your_token -p 3000:3000 ContribLens
```

Then visit `http://localhost:3000/svg/your-username`.

## Rules applied

Files are **excluded** from language attribution when they match any of these rules:

- **Lock files** ‚Äî `package-lock.json`, `yarn.lock`, `Gemfile.lock`, `Cargo.lock`, etc.
- **Generated code** ‚Äî files with `@generated`, `Code generated by`, `DO NOT EDIT` headers
- **Generated paths** ‚Äî `__generated__/`, `*.pb.go`, `*.min.js`, migration files, etc.
- **Vendor code** ‚Äî `vendor/`, `third_party/`, `node_modules/` (if somehow committed)
- **Minified files** ‚Äî files with lines exceeding 500 characters
- **Binary files** ‚Äî images, fonts, compiled artifacts
- **Rename-only commits** ‚Äî no actual code change
- **Merge commits** ‚Äî bot-generated merge noise
- **Documentation-only** ‚Äî `.md`, `.txt`, `.rst` files
- **Formatting-only** ‚Äî whitespace/indent-only diffs
